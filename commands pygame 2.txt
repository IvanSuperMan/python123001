pygame.image.load().convert() - загрузка изображения в игру и конвертация его для более быстрой обработки;
pygame.transform.scale() - изменение размера изображения;
pygame.Rect() - определяем прямоугольную область для обработки коллизий (например вокруг шарика);
.colliderect() - это метод для определения столкновений между объектами;
.collidelist() - это метод для определения с каким из объектов в списке произошло столкновение;
pygame.draw.circle() - рисуем круг на экране.
